## Log in to the Thunder Console

1. Open your browser and navigate to the Thunder console URL (e.g., `https://your-thunder-instance.com`).
2. Enter your admin credentials and log in.

## Navigate to the Users Section

1. From the Thunder console dashboard, locate the **Users** section in the sidebar menu.
2. Click on **Users** to view the list of existing users.

## Create a New User

1. Click the **Create User** button.
2. Fill in the required details:
   - **Username**: Enter a unique username for the user.
   - **Email**: Provide the user's email address.
   - **Password**: Set a secure password for the user.
3. Assign roles to the user by selecting from the available roles.
4. Click **Save** to create the user.

## Test User Login

1. Log out of the admin account.
2. Attempt to log in using the newly created user's credentials.
3. Verify that the user has access to the appropriate resources based on their assigned roles.
